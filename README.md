# Skincare App

Mobile skincare tracking app built with Expo, React Native, TypeScript, and Expo Router.

## Requirements

- Node.js 24
- npm 11
- Xcode for the iOS simulator

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
npm run ios
npm run android
npm run web
```

Use `npm run dev:clear` when the Metro cache needs to be reset.

## Quality

```bash
npm run format
npm run lint
npm run typecheck
npm run test
npm run doctor
npm run check
```

`npm run check` is the local validation command to run before committing or opening a pull request.
